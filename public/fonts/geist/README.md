# Geist Font Files

This directory should contain the following WOFF2 font files:

- Geist-Regular.woff2 (weight: 400)
- Geist-Medium.woff2 (weight: 500)
- Geist-SemiBold.woff2 (weight: 600)
- Geist-Bold.woff2 (weight: 700)

## How to Generate Font Files

If the font files are missing, you can generate them using:

1. **Transfonter** (https://transfonter.org/):
   - Upload the Geist font files (TTF/OTF)
   - Select "WOFF2" format only
   - Download and extract files to this directory

2. **Font Squirrel Webfont Generator** (https://www.fontsquirrel.com/tools/webfont-generator):
   - Upload the Geist font files
   - Select "WOFF2" format
   - Download and extract files to this directory

## Font Source

Geist font can be obtained from:
- Vercel's GitHub: https://github.com/vercel/geist-font
- Or use a similar modern sans-serif font if Geist is not available
